  • createLoginUrl(scopes:provider:)

    Create a URL that can be opened in a web browser to request permission to access a user’s account. If a user authorizes the application, the browser will redirect to the provided redirectUri that was passed to init. Included will be a code query string parameter for use with getAccessToken(code:completionHandler:).
